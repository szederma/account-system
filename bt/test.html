Technical Specifications

Explore the hardware and capabilities that power the Distanzuino

Hardware

  • Processor Dual-core ARM Cortex-M7, 480MHz
  • Distance Sensors 4x ToF sensors, 0.1mm precision
  • Motion Tracking 9-axis IMU (gyro, accel, compass)
  • Display 1.3" OLED, 128x64 resolution
  • RGB Lighting 24 programmable RGB LEDs
  • Battery 3000mAh Li-Po, up to 16 hours
  • Charging USB-C PD, 2 hours full charge
  • Dimensions 150mm x 90mm x 25mm
  • Weight 210g

Connectivity

  • USB USB-C, class-compliant MIDI
  • Bluetooth BLE 5.2, MIDI over BLE
  • WiFi 802.11ac for OTA updates
  • Analog Audio 3.5mm stereo output
  • Expansion I2C port for accessories
  • MIDI DIN-5 MIDI I/O ports
  • Pedal Input 2x 1/4" TRS expression inputs
  • Sync CV/Gate input and output

Software

  • OS Custom real-time OS
  • Sound Engine 64-bit floating point DSP
  • Latency 0.9ms sensor to sound
  • Presets 512 user-savable configurations
  • Mobile App iOS and Android compatible
  • DAW Integration VST3, AU, AAX plugins
  • Updates Over-the-air firmware updates
  • Developer API Open SDK for extensions
